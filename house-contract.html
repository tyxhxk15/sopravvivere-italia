정착 가이드

집 계약 체크리스트

이탈리아에서 집을 구할 때 반드시 확인해야 하는 항목들을 정리했습니다.

```

1. 계약 전 확인사항

  • 집 주소와 실제 위치 확인
  • 학교·직장까지 이동시간 확인
  • 주변 마트·약국·정류장 확인
  • 난방 방식 확인
  • 인터넷 설치 가능 여부 확인

2. 계약서 확인

  • 계약 기간 확인
  • 보증금(Deposìto) 금액 확인
  • 월세 납부일 확인
  • 계약 해지 조건 확인
  • 계약 등록 여부 확인

3. 공과금 확인

  • 전기 포함 여부
  • 가스 포함 여부
  • 수도 포함 여부
  • 관리비(Spese condominiali) 포함 여부
  • 인터넷 비용 포함 여부

4. 입주 전 체크

  • 가구 및 가전 상태 촬영
  • 벽·바닥 하자 촬영
  • 수도 및 난방 작동 확인
  • 열쇠 개수 확인
  • 계량기 사진 보관

5. 주의사항

  • 현금 선입금을 요구하면 주의
  • 계약서 없이 입주하지 않기
  • 계약 등록 여부 반드시 확인
  • 보증금 반환 조건 서면 확인
```
